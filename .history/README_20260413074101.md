# Sheet Builder

Build reusable CRUD web apps starting from exported Google Sheets files.

## Current pipeline

Input:
- `.xlsx`
- `.html` optional

Pipeline:
1. parse spreadsheet structure
2. merge visual metadata from HTML
3. build `parsed.schema.json`
4. build `builder_state.json`
5. generate:
   - `codice.gs`
   - `docs/index.html`
   - `docs/viewer.html`
